# cancer-ml-project
Beginner ML project on cancer classification using Python
# Cancer Classification using Machine Learning

This project demonstrates a beginner-level machine learning workflow
applied to a biomedical dataset.

## Tools Used
- Python
- Pandas
- scikit-learn
- Google Colab

## Description
The model was trained to classify cancer samples using biological features.
Basic data preprocessing, model training, and evaluation were performed.

## Outcome
The model achieved good accuracy, demonstrating how ML can be applied
to biological and drug discovery problems.
